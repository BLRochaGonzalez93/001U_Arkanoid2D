## Contenido

- Demo funcional de un nivel inspirada en Arkanoid.
- Movimiento horizontal de la barra controlada por el jugador.
- Pelota con físicas 2D y rebotes.
- Ladrillos con distintos niveles de durabilidad.
- Sistema de puntuación.
- Sistema de vidas y condición de derrota.
- Partículas en colisiones.
- Aceleración progresiva de la pelota.

## Cómo ejecutar o instalar

1. Abrir la página de GitHub Releases enlazada arriba.
2. Descargar el archivo de la build disponible en la sección **Assets**.
3. Extraer el `.zip` en una carpeta local.
4. Ejecutar el archivo `.exe` del juego.
5. Mantener el `.exe` dentro de la carpeta extraída, junto con los archivos internos generados por Unity.

## Requisitos aproximados

- Sistema operativo: Windows 10 o superior.
- Arquitectura: 64 bits.
- GPU compatible con DirectX 11.
- Espacio libre suficiente para descargar y extraer la build.
- Hardware capaz de ejecutar juegos desarrollados en Unity.

## Problemas conocidos / notas

- La build representa una versión de prototipo/proyecto jugable.
- Puede aparecer aviso de Windows al no estar firmada digitalmente.
- El balance de velocidad y rebotes puede no representar una versión final.
